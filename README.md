# BAdle

A simple wordle game featured [Blue Archive](https://www.youtube.com/channel/UCsrnDYrkovQhCCE8kwKcvKQ) theme.

## Data Source

[Schale Wiki](https://schale.gg)

## Tech Stacks

- React
- Bun
- Vite
- TypeScript
- Sass
- MobX
- Radix UI

### Crawler

- Crawlee
- Playwright
